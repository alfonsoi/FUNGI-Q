FUNGI Q

This repository includes the AI training dataset used to develop computer vision models for the system. These models are deployed on the Arduino UNO Q using Edge Impulse, enabling real-time edge inference.

It also contains project planning materials and documentation of the interaction between the MPU and MCU of the Arduino UNO Q, developed through App Lab.

Additionally, this repository contains the CAD design of the modular frame based on aluminum extruded profiles (Bosch-type). The structure is designed for scalability and easy assembly of multiple growing units.

Custom connectors and attachments will be 3D printed to integrate the growing tubes, sensors, and electronic components into the frame. This allows flexible, low-cost, and adaptable deployment in urban environments.

This enables a scalable and replicable system for distributed urban mushroom production.
