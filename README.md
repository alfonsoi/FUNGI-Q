# FUNGI-Q

**CADs files**

 #This file contains the CAD design of the modular frame based on aluminum extruded profiles (Bosch-type). The structure is designed for scalability and easy assembly of multiple growing units.

Custom connectors and attachments will be 3D printed to integrate the growing tubes, sensors, and electronic components into the frame. This allows flexible, low-cost, and adaptable deployment in urban environments.

This enables a scalable and replicable system for distributed urban food production
